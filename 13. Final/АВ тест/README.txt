Новое решение на сайте — назойливый чат с менеджером.

Шаг 1. Открою файл с данными и изучу общую информацию
Шаг 2. Подготовлю данные
	- переименую столбцы и данные для удобства
	- поменяю формат даты и времени, добавлю отдельный столбец дат
	- проверю на наличае дубликатов
Шаг 3. Изучу и проверю данные
	- посчитаю сколько в среднем событий приходится на пользователя
	- проверю, что у меня есть пользователи из всех трёх экспериментальных групп
Шаг 4. Изучаю воронку событий
	- посмотрю, какие события есть в логах, как часто они встречаются
	- посчитаю долю пользователей, которые хоть раз совершали событие
	- по воронке событий посчитаю, какая доля пользователей проходит на следующий шаг воронки (от числа пользователей на предыдущем)
	- определю какая доля пользователей доходит от первого события до оплаты
Шаг 5. Изучу результаты эксперимента
	- проверю корректно ли поделены группы
	- сравню результаты с каждой из контрольных групп в отдельности по каждому событию
	- проверю свои результаты при разных уровнях статистической значимости
Шаг 6. Общий вывод
